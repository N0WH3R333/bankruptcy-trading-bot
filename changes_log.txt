ИНСТРУКЦИИ ПО НАСТРОЙКЕ:
------------------------
1. Установить зависимости: pip install -r requirements.txt
2. Создать .env файл из env_example.txt
3. Заполнить API ключи и токены
4. Настроить контакты в config.py
5. Запустить: python bot.py

ПРИМЕЧАНИЯ:
-----------
- Все промпты легко редактируются в prompts.py
- База данных создается автоматически
- Логи сохраняются в bot.log и admin_bot.log
- Статистика доступна через админ-панель
